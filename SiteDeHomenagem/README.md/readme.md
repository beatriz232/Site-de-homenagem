# 💖 Site de Homenagem Romântica

Um site interativo desenvolvido como homenagem especial, combinando amor, criatividade e programação.

Este projeto foi criado para eternizar momentos, memórias e sentimentos através de uma experiência visual romântica e envolvente.

---

## ✨ Sobre o Projeto

Este site funciona como um espaço digital de memórias, contendo:

- 🔐 Tela protegida por senha
- ⏳ Contador desde 07/04/2025
- 💞 Animação de corações em canvas
- 🎵 Player de música personalizado
- 🖼 Slideshow de fotos
- 🎬 Vídeo especial
- 🧩 Páginas interativas (quiz, puzzle, linha do tempo, memórias)

O objetivo foi criar uma experiência emocional e visualmente agradável, utilizando apenas tecnologias web.

---

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3 (animações, glassmorphism, responsividade)
- JavaScript (interatividade, animações, player de música)
- Canvas API (efeito de corações animados)
